React State & Events

7. State

What is State?

State is a built-in React object used to store data that can change over time and affect how the UI behaves.

  • ✔ State is mutable
  • ✔ When state changes → component re-renders
  • ✔ State is local to a component

Static Example

function Counter() {
  return <h1>0</h1>;
}

This value is static. To make it dynamic, we use state.

useState Hook

useState is a React Hook used to create and manage state in functional components. 

import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);
  return <h1>{count}</h1>;
}
  • count current state value
  • setCount function to update state
  • 0 initial value

Updating State

Never update state directly.

❌ Wrong:

count = count + 1;

✅ Correct:

setCount(count + 1);

Example with Button

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <>
      <h1>{count}</h1>
      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
    
  );
}

Updating State Using Previous Value

setCount(prevCount => prevCount + 1);

React batches state updates for performance, so using the previous value ensures accuracy.

State vs Props

Feature State Props
Mutability Mutable Immutable
Ownership Inside component Passed from parent
Purpose Manage data Pass data
Updated by useState Parent only 
<Counter initialValue={5} />

function Counter({ initialValue }) {
  const [count, setCount] = useState(initialValue);
}

8. Events

Handling Events

React events use camelCase and JSX syntax. 

<button onClick={handleClick}>Click Me</button>

function handleClick() {
  alert("Button clicked!");
}

onClick Event

<button onClick={() => console.log("Clicked!")}>
  Click
</button>

✔ Always pass a function

❌ Wrong:

onClick={handleClick()}

onChange Event

function Input() {
  const [text, setText] = useState("");

  return (
    <input
      type="text"
      value={text}
      onChange={(e) => setText(e.target.value)}
    />
  );
}

e is a Synthetic Event (React wrapper around native events).

Passing Parameters to Events

Method 1: Arrow Function (Recommended)

<button onClick={() => handleDelete(5)}>
  Delete
</button>

function handleDelete(id) {
  console.log(id);
}

Method 2: bind()

<button onClick={handleDelete.bind(null, 5)}>
  Delete
</button>

Method 3: Event + Parameter

<button onClick={(e) => handleClick(e, "Hello")}>
  Click
</button>

function handleClick(event, message) {
  console.log(message);
}
